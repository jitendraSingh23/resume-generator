## Run Locally

1. Clone the project

```bash
git clone https://github.com/jitendraSingh23/Lorem-ipsum-generator.git
```

2. Go to the project directory

```bash
cd resume generator
```
3. create .env

```bash
npm install
```
4. create .env

```bash
npm run dev
```

5. Try to access app on your [localhost:5173](http://localhost:5173/).
